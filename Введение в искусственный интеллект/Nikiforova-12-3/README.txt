Компиляция:
        chmod +x compile.sh
        ./compile.sh

Запуск:
        ./a.out

Каждый запуск будет давать разный результат, т.к. в программе используются случайные начальные веса.

Построить графики анимации процесса обучения можно двумя способами:
1. В отдельном окне:
        gnuplot -p plot_window.gnu
Чтобы преключить кадр нужно будет нажать левую кнопку мыши на графике в открывшемся окне.
Для повторной анимации необходимо запустить скрипт заново.

2. Вся анимация в отдельном gif-файле:
        gnuplot plot_gif.gnu
После запуска нужно будет открыть файл results.gif, анимация зациклена.

На графике в левом нижнем углу отображается номер итерации изменения весов 
(не для конкретного обучающего примера, а в целом).

Иногда на графике могут быть не видны первая или вторая итерация, т.к. процесс начинается со случайных 
весов и линия может не попасть в область видимости терминала.

Разными цветами на графике обозначены точки, принадлежащие к разным классам.
Маркеры-круги обозначают точки обучающей выборки, маркеры-* - тестовой.
